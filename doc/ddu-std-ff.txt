FUNCTIONS					*ddu-ff-ui-functions*

						*ddu#ff#ui#do_action()*
ddu#ff#ui#do_action({action-name}[, {params}])
		Fire {action-name} action with {params}.  You can find the
		actions list in |ddu-ff-ui-actions|.
		{args} behavior depends on {action-name}.
		Note: It cannot be used in the filter window. >

------------------------------------------------------------------------------
ACTIONS						*ddu-ff-ui-actions*

						*ddu-ff-ui-action-itemAction*
itemAction
		params:
			{name}: action name

		Close the UI window and fire {name} action for items.
		You can find the actions list in item's kind documentation.
		If {name} is empty, "default" will be used.

		Note: You cannot mix multiple kinds/sources.

					*ddu-ff-ui-action-openFilterWindow*
openFilterWindow
		Open the filter window.

					*ddu-ff-ui-action-preview*
preview
		Preview the item in |preview-window|.

					*ddu-ff-ui-action-refreshItems*
refreshItems
		Refresh the items.

					*ddu-ff-ui-action-toggleSelectItem*
toggleSelectItem
		Toggle selected item.

						*ddu-ff-ui-action-quit*
quit
		Quit the UI window.

					*ddu-ff-ui-action-updateOptions*
updateOptions
		params:
			{option-name}: option name and value

		Update current options and redraw the UI.
		Refer to |ddu-options| about options.

PARAMS						*ddu-ff-ui-params*

				    	*ddu-ff-ui-param-autoResize*
autoResize	(string)
		Auto resize the window height automatically.

		Default: v:false

						*ddu-ff-ui-param-cursorPos*
cursorPos	(number)
		Select {number} candidate.

		Default: 0

				    	*ddu-ff-ui-param-displaySourceName*
displaySourceName	(string)
		Display source name in the buffer. Following values are
		available:

		"long": display source name
		"no": does not display

		Default: "no"

				    	*ddu-option-filterFloatingPosition*
filterFloatingPosition	(string)
		Specify the filter floating window position as {position}.

		If {position} is set to "bottom", the filter will be displayed
		at the bottom.
		If {position} is "top", the filter will be displayed at the top.

		Note: To use it, you need to use neovim and
		|ddu-option-filterSplitDirection|must be "floating".

		Default: "bottom"

				    	*ddu-option-filterSplitDirection*
filterSplitDirection	(string)
		Specify the filter window direction as {direction}.

		When {direction} is "floating", the filter uses neovim
		floating window. If statusline is at the bottom, the filter
		overwrites statusline. Otherwise, it is located one line below
		statusline.

		Note: To use "floating", you need to use neovim.

		Default: "botright"

					*ddu-ff-ui-param-previewFloating*
previewFloating	(boolean)
		Use floating window in |ddu-ff-ui-action-preview|.

		Default: false

						*ddu-ff-ui-param-previewHeight*
previewHeight	(number)
		Set the height of the |preview-window| in
		|ddu-ff-ui-action-preview|.
		If |ddu-ff-ui-param-previewFloating|, set the height of the
		floating window.

		Default: 10

					*ddu-ff-ui-param-previewVertical*
previewVertical	(boolean)
		Use vertical preview instead of horizontal preview in
		|ddu-ff-ui-action-preview|.

		Default: false

						*ddu-ff-ui-param-previewWidth*
previewWidth	(number)
		Set the width of the |preview-window| in
		|ddu-ff-ui-action-preview|.
		If |ddu-ff-ui-param-previewFloating|, set the width of the
		floating window.

		Default: 40

				    		*ddu-ff-ui-param-prompt*
prompt		(string)
		Specify the prompt in the filter window.
		Note: It must be one or two width characters.

		Default: ""

				    		*ddu-ff-ui-param-split*
split		(string)
		Specify split mode. Following values are available:

		"horizontal": horizontal split
		"vertical": vertical split
		"floating": use neovim floating window feature
		"no": no split

		Default: "horizontal"

				    		*ddu-ff-ui-param-startFilter*
startFilter	(boolean)
		If it is true, the filter window is opened automatically.

		Default: v:false

						*ddu-ff-ui-param-winCol*
winCol		(number)
		Set the column position of the window if
		|ddu-ff-ui-param-split| is "floating".

		Default: &columns / 4

						*ddu-ff-ui-param-winHeight*
winHeight	(number)
		Set the height of the window if |ddu-ff-ui-param-split| is
		"horizontal".
		if |ddu-ff-ui-param-split| is "floating",
		set the height of the floating window.

		Default: 20

						*ddu-ff-ui-param-winRow*
winRow		(number)
		Set the row position of the window if |ddu-ff-ui-param-split|
		is "floating".
		If it is 1, the filter window position is always top.

		Default: &lines / 2 - 10

						*ddu-ff-ui-param-winwidth*
winWidth	(number)
		Set the width of the window if |ddu-ff-ui-param-split| is
		"vertical".
		if |ddu-ff-ui-param-split| is "floating", set the width of
		the floating window.

		Default: &columns / 2

EXAMPLES					*ddu-ff-ui-examples*
>
	autocmd FileType ddu-ff call s:ddu_my_settings()
	function! s:ddu_my_settings() abort
	  nnoremap <buffer><silent> <CR>
	  \ <Cmd>call ddu#ui#ff#do_action('itemAction')<CR>
	  nnoremap <buffer><silent> <Space>
	  \ <Cmd>call ddu#ui#ff#do_action('toggleSelectItem')<CR>
	  nnoremap <buffer><silent> i
	  \ <Cmd>call ddu#ui#ff#do_action('openFilterWindow')<CR>
	  nnoremap <buffer><silent> q
	  \ <Cmd>call ddu#ui#ff#do_action('quit')<CR>
	endfunction

	autocmd FileType ddu-ff-filter call s:ddu_filter_my_settings()
	function! s:ddu_filter_my_settings() abort
	  inoremap <buffer><silent> <CR>
	  \ <Esc><Cmd>close<CR>
	  nnoremap <buffer><silent> <CR>
	  \ <Cmd>close<CR>
	endfunction
